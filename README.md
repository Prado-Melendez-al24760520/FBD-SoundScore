# FBD-SoundScore
FBD-SoundScore planea un sistema de reseñas de albums o canciones 

## Motivacion 
La música es una de las formas de expresión más universales y personales, y cada oyente tiene opiniones distintas sobre canciones y álbumes.Pero, muchas veces estas reseñas quedan dispersas en redes sociales o plataformas.
